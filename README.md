# 🎮 Python Hangman Game

A simple terminal-based Hangman game built using Python.

This project was created to practice:
- Python basics
- loops and conditions
- functions
- file handling
- modular programming
- Git and GitHub workflow

---

# 🚀 Features

✅ Random word generation  
✅ 300+ word support  
✅ Hangman ASCII art stages  
✅ Replay option  
✅ Score system  
✅ Duplicate guess checking  
✅ Multiple Python files for clean structure
